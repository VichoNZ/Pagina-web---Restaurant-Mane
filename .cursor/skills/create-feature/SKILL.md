# Crear funcionalidad nueva

Cuando el usuario pida agregar una funcionalidad:

1. Identifica los archivos mínimos a modificar.
2. Explica el plan antes de editar.
3. Haz un cambio pequeño primero.
4. No reescribas todo el proyecto.
5. Mantén la arquitectura actual.
6. Al final entrega:
   - Archivos modificados.
   - Qué se agregó.
   - Cómo probarlo.
